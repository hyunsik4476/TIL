# APS

## 비트연산

* ![캡처](README.assets/캡처.PNG)
* 비트연산과 마스킹(`& 0xff`)
* 마스킹에 의해 첫 8자리 이후의 비트는 날아감
* `| 0000100..000` :  원하는 자리 1로 set
* `& 1110111..111` : 원하는 자리 0으로 clear
